Fire-S Sprint 111.3 - Analytics Isolation Hard Fix

Purpose:
- Keep the normal inspection / project pages operational-only.
- Statistical, comparison, trend, history and executive intelligence panels no longer appear in the page flow.
- Analytics opens as a separate overlay workspace from the Analytics button.

Fixes:
- Removes Inspection History panel from operational page flow.
- Removes Inspection Comparison panel from operational page flow.
- Removes Trend Analytics panel from operational page flow.
- Removes Executive Intelligence / Management View panels from operational page flow.
- Moves the Analytics workspace out of the Project Details flow and displays it as a separate overlay.
- Adds Start New Inspection to the More menu so the workflow is still accessible without showing the full history panel on the inspection page.

Result:
Inspection pages now show only operational content such as project details, checklist, photos, action register and report workflow.
Analytics remains available through the Analytics button only.
