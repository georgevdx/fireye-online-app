Purpose:
When an existing inspection is opened, Fire-S now shows an Inspection Open Choice:
1. Archive as History + Start New Inspection
2. Continue / Edit Current Inspection
3. Delete Inspection - disabled for data safety

Delete is intentionally disabled in this gate until it is rebuilt safely.
